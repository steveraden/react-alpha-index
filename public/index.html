React Alphabetical Index

This site is a demonstration of react-alphabetic-index an open source alphabetic index project created in React.js. Click any letter to trigger an AJAX query with that letter as a parameter. Responses are then rendered in a data grid. This project also utilized Sinatra, Postgresql Heroku for a simple API. Bootstrap is utilized to a passable look and feel. The data set is the Webster Signature database. More information on that project is available at the right.

About the Webster Signature Database

This demo utilizes the Webster Institute's public database of the signature of a astronomy instruments in various collections world-wide.To learn more about the database or the database, please visit the following links.
